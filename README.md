# מחשבון שכר ועלות מעביד — ישראל 2025

מחשבון שכר מקצועי לחישוב מס הכנסה, ביטוח לאומי, נקודות זיכוי ועלות מעביד בישראל.

## תכונות

- חישוב ברוטו → נטו ונטו → ברוטו
- חישוב נקודות זיכוי אוטומטי לפי פרטי עובד
- מדרגות מס הכנסה 2025
- ביטוח לאומי ומס בריאות (עובד ומעסיק)
- פנסיה, פיצויים וקרן השתלמות
- עלות מעביד מלאה
- סיכום שנתי
- מצב כהה (Dark Mode)
- שמירת נתונים ב-localStorage
- תמיכה מלאה במובייל

## הרצה

### שיטה 1 — ישירות מהדפדפן (פשוטה ביותר)

פתח את `index.html` ישירות בדפדפן. האפליקציה תעבוד עם נתוני מס מוטמעים.

> **שים לב:** קובצי JSON חיצוניים לא ייטענו בשיטה זו בשל מגבלות אבטחה של הדפדפן (CORS מ-file://). נתוני ברירת המחדל המוטמעים זהים לנתונים בקובצי ה-JSON.

### שיטה 2 — שרת HTTP מקומי (עם קובצי JSON חיצוניים)

לטעינת קובצי JSON חיצוניים (ומאפשרת עדכון נתוני מס ללא שינוי קוד), הרץ שרת מקומי:

**Python (מגיע עם כל מחשב):**
```bash
cd "Employer cost"
python -m http.server 8080
```
ואז פתח: http://localhost:8080

**Node.js:**
```bash
npx http-server "Employer cost" -p 8080
```

**VS Code:** התקן את תוסף "Live Server" ולחץ על "Go Live".

## עדכון נתוני מס

כל נתוני המס נמצאים בתיקיית `data/`:

| קובץ | תוכן |
|------|-------|
| `tax-brackets.json` | מדרגות מס הכנסה חודשיות |
| `national-insurance.json` | שיעורי ביטוח לאומי (עובד ומעסיק) |
| `health-tax.json` | שיעורי מס בריאות |
| `credit-points.json` | שווי וכללי נקודות זיכוי |

לעדכון שנתי — שנה את הערכים בקובצי JSON בלבד, ללא שינוי קוד.
