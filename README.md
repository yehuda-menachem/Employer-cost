# 💼 מחשבון שכר ועלות מעביד — ישראל (2025 - 2026)

מערכת אינטראקטיבית ומקצועית לחישוב מקיף של שכר ברוטו ונטו, נקודות זיכוי, הפרשות סוציאליות ועלות מעביד מלאה בישראל. המערכת מותאמת באופן מלא לחוקי המס והתקנות של שנות המס **2025 ו-2026**.

🚀 **[לצפייה בגרסה החיה (GitHub Pages)](#-פריסה-והעלאה-לאוויר-github-pages)**

---

## 🌟 תכונות עיקריות

- **דו-כיווניות מלאה (Gross ⇄ Net):** 
  - חישוב **ברוטו לנטו** (פירוט מלא של ניכויים והפרשות).
  - חישוב **נטו לברוטו** (מבוצע באמצעות אלגוריתם חיפוש בינארי מהיר ומדויק להתכנסות מושלמת).
- **מנוע נקודות זיכוי מתקדם:**
  - חישוב אוטומטי מבוסס מגדר, תושבות, מצב משפחתי וגילאי ילדים.
  - תמיכה בהטבות מיוחדות: חיילים משוחררים, עולים חדשים ונקודות זיכוי מותאמות אישית.
- **הנחת יישובים (סעיף 11):** חישוב הנחת מס מבוססת יישובים מזכים באופן דינמי.
- **הפרשות סוציאליות ועלות מעסיק:**
  - הפרשות פנסיה (עובד ומעביד) ופיצויים.
  - קרן השתלמות (עובד ומעביד) כולל התחשבות בתקרות פטור ממס.
  - חישוב עלויות מעביד מלאות (כולל ביטוח לאומי מעסיק).
- **חוויית משתמש פרימיום:**
  - ממשק רספונסיבי מלא (Mobile Friendly) בעל עיצוב נקי ומודרני.
  - תמיכה ב**מצב כהה (Dark Mode)** מובנה עם מעברים חלקים.
  - שמירת נתוני החישוב האחרון ב-`localStorage` לטעינה מהירה.
  - אפשרות להדפסת תלוש דמוי-פירוט שכר בצורה אסתטית.
  - מודל מובנה לשליחת משוב מפורט במייל (כולל העתקת נתוני החישוב ללוח).

## 📂 מבנה הפרויקט

הפרויקט בנוי בצורה מודולרית ללא תלות בספריות חיצוניות (Vanilla JS):

```text
Employer cost/
├── index.html                  # ממשק המשתמש הראשי (HTML5 סמנטי ונגיש)
├── style.css                   # עיצוב מודרני, משתני CSS (CSS Variables) ותמיכה ב-Dark Mode
├── app.js                      # מנהל המצב (State), האזנה לאירועים ואינטגרציה
│
├── calculators/                # מנועי החישוב (מודולריים ומנוטרלים מאירועי DOM)
│   ├── credits.js              # חישוב נקודות זיכוי ושווין הכספי
│   ├── incomeTax.js            # מדרגות מס הכנסה וחישוב מס אפקטיבי
│   ├── insurance.js            # חישוב ביטוח לאומי ומס בריאות (עובד/מעביד)
│   ├── pension.js              # חישוב הפרשות לפנסיה וקרן השתלמות
│   ├── employerCost.js         # סיכום עלויות המעסיק
│   ├── grossFromNet.js         # מנוע ההתכנסות מנטו לברוטו
│   ├── qualifyingSettlement.js # חישוב הטבות מס ליישובים מזכים
│   └── pensionTaxCredit.js     # חישוב הטבות מס בגין הפרשות לפנסיה
│
├── data/                       # קובצי הנתונים (מדרגות מס, תקרות ושיעורים לשנים 2025/2026)
│   ├── tax-brackets.json
│   ├── national-insurance.json
│   ├── health-tax.json
│   ├── credit-points.json
│   ├── qualifying-settlements.json
│   └── tax-parameters.json
│
└── utils/                      # ספריות עזר
    ├── formatters.js           # עיצוב מספרים, מטבעות (₪) ואחוזים
    └── validators.js           # בדיקת תקינות קלטים
```

## 🛠 הרצה מקומית

כדי לטעון את נתוני המס בצורה דינמית מקובצי ה-JSON, יש להריץ את הפרויקט תחת שרת מקומי (בשל מגבלות אבטחה של הדפדפן - CORS על פרוטוקול `file://`).

> 💡 **גיבוי מובנה (Fallback):** במקרה של הרצה ישירה ללא שרת (פתיחה כפול קליק על `index.html`), המערכת תשתמש בנתוני גיבוי פנימיים המוטמעים בקוד המקור. נתוני הגיבוי זהים לחלוטין לנתונים בקובצי ה-JSON.

### אפשרויות להרצה מקומית:

**1. באמצעות Python (זמין ברוב המערכות):**
```bash
python -m http.server 8080
```
לאחר מכן פתחו בדפדפן: `http://localhost:8080`

**2. באמצעות Node.js:**
```bash
npx http-server -p 8080
```

**3. באמצעות VS Code:**
התקינו את התוסף **Live Server** ולחצו על **Go Live** בפינה הימנית התחתונה.

## 🚀 פריסה והעלאה לאוויר (GitHub Pages)

הפרויקט מותאם במלואו לפריסה מהירה בחינם ב-GitHub Pages ללא צורך בשום שלב בנייה (Build Step).

### שלבי הפריסה:
1. ודאו שכל קובצי הפרויקט (כולל תיקיות `data`, `calculators`, `utils` וקובץ `index.html`) דחופים (Pushed) למאגר ה-GitHub שלכם.
2. בהגדרות המאגר ב-GitHub, עברו ללשונית **Settings** -> **Pages** (בתפריט השמאלי).
3. תחת **Build and deployment**:
   - בחרו ב-**Deploy from a branch**.
   - בחרו בענף הראשי שלכם (`main` או `master`) ותיקיית השורש (`/ (root)`).
   - לחצו על **Save**.
4. האתר יהיה זמין בכתובת:  
   `https://<your-username>.github.io/<repository-name>/`

## ⚙️ עדכון נתוני מס (תחזוקה)

המערכת מתוכננת כך ששינויים שנתיים במדרגות המס או בערכי ביטוח לאומי אינם דורשים שינוי בקוד המקור. כל הנתונים נטענים מתוך קובצי ה-JSON שבתיקיית `data/`:

- `tax-brackets.json` - מדרגות המס השנתיות והחודשיות.
- `national-insurance.json` - אחוזי ותקרות ביטוח לאומי (עובד ומעסיק).
- `health-tax.json` - אחוזי מס בריאות לפי מדרגות ביטוח לאומי.
- `credit-points.json` - שווי נקודת זיכוי שנתית וחודשית.
- `qualifying-settlements.json` - רשימת היישובים המזכים, אחוזי ההנחה ותקרות הזכאות.

לעדכון הנתונים לשנים הבאות, פשוט יש לעדכן את קובצי ה-JSON הרלוונטיים.
